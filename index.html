Avaliação da Eficácia dos Testes Caixa Preta para a Detecção de Falhas de Software

Descrição da pesquisa

Este estudo busca somar como um recurso didático, no idioma português, e, a partir disso, promover uma perspectiva abrangente sobre diferentes abordagens, técnicas e estratégias para detectar falhas de software mediante a realização dos testes caixa preta, a partir de uma análise do estudo e os resultados da aplicação obtidos através da investigação de testadores e desenvolvedores. Sendo assim, as técnicas serão colocadas em avaliação na análise de diferentes sistemas.

Parecer Consubstanciado aprovado pelo Comitê de Ética em Pesquisa. Número: 7.236.121.

Instruções para Testadores

  • Acesse os links dos sistemas hospedados e explore suas funcionalidades.
  • Verifique os dados de entrada e saída.
  • Observe o comportamento do sistema.
  • Os testes caixa preta não necessitam de acesso ao código fonte, os autores contam com a franqueza dos testadores em realizar a investigação apenas por meio da interface hospedada dos sistemas.
